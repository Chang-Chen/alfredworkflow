# alfredworkflow
记录日常开发使用的alfredworkflow,
更多好玩的工具请前往[下载](https://www.packal.org/workflow)

# CodeVar.alfredworkflow
生成变量名，支持大小驼峰、常量、下划线，开发者必备的工作流

# GitClone.alfredworkflow
自动读取剪切板的git仓库地址，并clone到指定的目录

# Len().alfredworkflow
复制一段文本，计算其长度

# Visual Studio Code plus
学习了 [alexchantastic/alfred-open-with-vscode-workflow](https://github.com/alexchantastic/alfred-open-with-vscode-workflow) 的代码后，增加了对st和idea的支持

# YoudaoTranslator-3.1-alfredv3.alfredworkflow
快速有道翻译：[配置参考](https://github.com/wensonsmith/YoudaoTranslator)

# colors.alfredworkflow
快速在 HEX、RGB、HSL 及 CSS 命名颜色之间进行转换与预览

# css.alfredworkflow
把写好的css【内联样式】和【内部样式之间】互相转换
* 内联->内部：复制内联样式（不包含style=""，仅复制里面的样式内容），alfred命令 `style`
* 内部->内联：复制内部样式（不包含&lt;style>&lt;/style>，仅复制里面的样式内容），alfred命令 `inline`

# curl2local
可以将剪切板里的curl命令转换成请求本地127.0.0.1的命令

# encode-decode.alfredworkflow
Base64 编解码

# github.alfredworkflow
快速查找项目/用户

# ip-address.alfredworkflow
快速查询本地ip和公网出口ip，再也不用到[ipip](http://ipip.net)上去查询了

# jsonEditor
[一款chrome插件](https://github.com/sunzsh/chromeapp-jsonedit)的辅助工具，可以在chrome中快速打开json编辑器

# package-managers.alfredworkflow
搜索和管理 ‌25+ 种包管理器‌（如 npm、Homebrew、Docker、PyPI 等），无需切换浏览器或终端，
[最新版下载链接](https://github.com/willfarrell/alfred-pkgman-workflow)

# random_pic.alfredworkflow
根据尺寸 `宽x高` or `宽*高` 随机生成图片到桌面
